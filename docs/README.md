# 기능명세서
## Car
    - [ ] 차의 이름을 검사한다(1자 이상 5자 이하)
    - [ ] 차를 움직인다(멈춤, 전진)
## RacingGame
    - [ ] 게임을 시작한다
        - 자동차의 이름들을 입력받는다
        - 시도할 횟수를 입력받는다.
    - [ ] 게임을 실행한다.
        - 차를 움직인다.
        - 실행 과정을 출력한다.
    - [ ] 게임을 종료한다.
        - 경기 우승자를 출력한다(1명 이상일 수 있다.)
