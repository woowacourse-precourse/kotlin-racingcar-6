## 기능 목록

### 자동차 이름 입력

- [x] 안내 메세지 출력<br>
  · `경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)`
- [x] 자동차 이름 쉼표로 구분해 입력 받기
  <br>
- [x] 자동차 이름 유효성 검사

  1. 이름이 숫자와 문자인지 검사
  2. 이름의 글자 수가 1자 이상, 5자 이하인지 검사
  3. 플레이어가 2명 이상인지 검사
  4. 각 이름이 중복되지 않는지 검사

### 시도 횟수 입력

- [x] 시도 횟수 입력 안내 메세지 출력<br>
  · `시도할 횟수는 몇 회인가요?`<br>
- [x] 시도 횟수 입력 받기
- [x] 시도 횟수 입력 유효성 검사

  1. 입력값이 비어있지 않은 지 검사
  2. 입력값의 숫자 여부 검사
  3. 범위 1 이상 2147483647 이하 여부 검사

### 게임 실행

- [x] 실행 결과 메세지 출력 (최초 1번)<br>
  · `실행 결과`

- [x] 0에서 9 중 무작위 선택한 값이 4 이상이면 자동차를 전진

- [x] 각 반복 시점에서 진행 상황 출력하기<br>
  · `pobi : --`<br>
  · `woni : ----`<br>
  · `jun : ---`


- [x] 최종 우승자명 출력 (공동 우승일 경우 쉼표로 구분)<br>
  · `최종 우승자 : pobi, jun`
