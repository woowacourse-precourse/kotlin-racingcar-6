<기능 구현 목록>

1. 게임 시작과 동시에  문구 출력
    - “경주할 자동차 이름을 입력하세요.(이름은 쉼표 (,) 기준으로 구분)"
2. 사용자에게 쉼표를 구분으로 하는 입력 값을 받음
    - 이름을 5자 이하로 강제하게 함
    - 3개를 받음
3. 사용자에게 시도할 횟수 입력 값을 받음
    - “시도할 횟수는 몇 회인가요?” 문구 출력함과 동시에 입력값 받기
4. 0과 9 사이에서 무작위 값을 구한 후 4 이상을 경우 전진한다(+1)
5. 각 차수별 실행 결과를 출력
6. 게임을 종료
    - 단독 우승자 혹은 공동 우승자 안내 문구를 출력한다.
