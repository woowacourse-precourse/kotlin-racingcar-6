### 🗂 기능 구성하기
- n대의 자동차 생성
    - 정확한 이름 입력 -> 게임 진행
    - 사용자가 잘못된 값을 입력할 경우 -> 게임 종료
- 총 시도 횟수 생성
    - 정확한 숫자 입력 -> 자동차 경주 시작
    - 사용자가 잘못된 값을 입력할 경우 -> 게임 종료
- 총 시도 횟수만큼 반복
    - 각 자동차의 무작위 값 생성
        - 무작위 값이 4 이상인 경우 : 전진("-")
        - 무작위 값이 4 미만인 경우 : 정지("")
    - 결과 출력 (자동차 이름 : 전진 상태)
- 최종 우승자 출력
