### 🚀 기능 요구 사항

- 자동차 이름을 입력받는다.
- 시도할 횟수를 입력받는다.
- 자동차별로 이동할 수 있는 임의의 값을 계산한다. 
- 자동차별로 이동한 거리를 출력한다.
- 우승자를 출력한다.

## 🚀 기능 목록

### Application

- [ ] 자동차 경주 게임을 시작한다.

### RacingGame

- [ ] 자동차 생성하기
- [ ] 경주 횟수 만큼 출력하기
- [ ] 결과를 출력하기

### Car
- [ ] 이름 입력 받기
- [ ] 이동 횟수 입력 받기

### Computer
- [ ] 우승자 판단

### 예외 처리

- 자동차 이름 예외
  - 자동차 이름이 5자 초과일 때
  - 이름이 비어있을 때
- 횟수 예외
  - 횟수는 숫자여야 한다.
  - 0보다 큰 숫자여야 한다.
