# 구현할 기능 목록

### 사용자 입력 객체

1. 자동차 이름 입력
   - 입력 안내 메시지 출력
   - 잘못 입력했을 경우 `IllegalArgumentException` 발생
     - string에 쉼표가 포함되어 있지 않은 경우
       -> 자동차 `경주` 이므로 최소 2대 이상 입력할 것이라 가정
     - 각 이름이 5자 이하가 아닌 경우
     - 자동차 이름이 중복된 경우


2. 시도 횟수 입력
   - 입력 안내 메시지 출력
   - 숫자가 아닐 경우 `IllegalArgumentException` 발생

---
### 자동차 객체

1. 자동차 객체
   - 이름과 전진 상황 상태를 가진 자동차 객체 


2. 자동차 전진 상황 저장

---
### 전진 여부 판단 객체

- 전진 여부 결정
   - 0과 9 사이의 무작위 값 생성
   - 해당 무작위 값이 4 이상인지 확인

---

### 출력 객체

1. 각 자동차 객체로부터 전진 상태를 받아서 진행 상황 출력
2. 경주 결과로 가장 많이 전진한 자동차 목록 출력

---

### 전체 자동차 관리 객체

1. 자동차 객체 리스트 생성
2. 전체 자동차 전진 상태 반환
3. 전진 여부 판단 객체의 메시지를 자동차 객체에 전달
   - 전진 상황 저장
