# 우아한테크코스 프리코스 2주차
+ **자동차 경주**

## 구현할 기능
### 자동차 클래스
    이름 반환 함수
    위치 관련 함수(위치 업데이트, 위치 반환)
    랜덤 숫자 생성
    숫자 판단

### View
    사용자의 이름 입력
    시도할 횟수 입력
    결과 출력

### 게임 클래스
    게임을 진행할 클래스
    이름별로 차 생성
    위치에 따라 우승자 판단

## 예외 상황
### 자동차 이름 입력 시
    이름이 5자 이상일 때 
    입력한 이름중에 중복되는 이름이 있을 시
    빈칸이 있을 때

### 시도할 횟수 입력 시 
    숫자가 아닌 값 입력 시
    공백일 때
