## 기능 목록
### 경주
- [x] 게임 시작
- [x] 최종 결과 출력

### 자동차
- [x] 전진 조건 구하기
- [x] 전진

### 진행자
- [x] 경주할 자동차 이름 입력 요구 문구 출력
- [x] 시도 횟수 질문 출력
- [x] 자동차 현재 위치 출력
- [ ] 우승자 안내 문구 출력

### 플레이어
- [x] 경주 할 자동차 이름을 입력
- [x] 시도할 횟수 입력

### 유효성 검사
- [x] 자동차 이름 유효성 검사
  - [x] 이름 5자 이하 아닐 때
  - [x] 공백 있을 때
  - [x] 1대만 입력 했을 때
  - [x] 같은 이름이 있을 때
  - [ ] 자동차 이름이 공란일 때
- [x] 시도 횟수 유효성 검사 
  - [x] 숫자 아닐 때
  - [x] 1보다 작을 때
