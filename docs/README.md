시작전

`경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)` 출력

자동차 입력

- [ ]  ,로 자동차를 나눈다.
- [ ]  이름은 5자 이내로만 가능하다

시도할 횟수 입력

- [ ]  int값만 입력받기

진행중

랜덤수 뽑기

- [ ]  (0~9 중에서 4 이상인 경우에만 전진 그 외는 stop)

자동차 마다 나온 수 만큼 진행시키기

- [ ]  go인 경우에 나온 수 만큼 출력

시도할 횟수만큼 반복시키기

종료

가장 많이 간 친구 뽑기

- [ ]  이름 출력하는데 두명이상일 경우 ,로 출력
<hr>

### 사용자

- [ ]  자동차(들)을 입력한다. 
- [ ]  몇번의 게임을 할지 정한다.

## 컴퓨터

- [ ]  0~9중 랜덤수를 뽑는다

## print

- [ ]  컴터가 뽑은 수 만큼 자동차(들)을 전진
